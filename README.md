# Stock Transaction Tracker
A command-line Java application that simulates stock purchases and sales using FIFO (First-In, First-Out) logic to calculate realized capital gains.

## Features
- Buy shares at a given price
- Sell shares using FIFO rules
- Calculates total realized capital gain
- Handles insufficient share warnings
  
## How to Run
javac StockLot.java StockTracker.java

java StockTracker
